## Περί της Συλλογής

*Περί Βίου και Λόγου* είναι μια σειρά φιλοσοφικών πραγματειών και δοκιμίων που διερευνούν τα μεγάλα υπαρξιακά ζητήματα του ανθρώπου: τον σκοπό, τον θάνατο, τη φιλία, την αρετή. Γραμμένα στην ελληνική γλώσσα, με αντλούν από την αρχαία ελληνική σκέψη και τη βιωματική εμπειρία.

> *«Μια διερεύνηση της ανθρώπινης εμπειρίας μέσα από τον διαχρονικό ελληνικό λόγο.»*
